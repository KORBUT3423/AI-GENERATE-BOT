# DamageBot — Telegram bot с Replicate Image-4 (расширенная версия)

## Описание
Бот принимает фото, позволяет выбрать damage-эффект, обрабатывает фото через Replicate (image-4) и отправляет результат. Имеет очередь задач, мультиязычный интерфейс, логирование и админ-команды.
